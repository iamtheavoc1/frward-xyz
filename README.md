# frward.xyz — public issue tracker and documentation

> **This is not the source code repository.** frward.xyz is proprietary, closed-source software. No source code is published here.

**frward.xyz** is a hosted AI router. Use this repository to report bugs, request providers, track roadmap progress, and read public documentation.

## What this repo is

- 🐛 Bug reports and issue tracking
- 📡 Provider requests
- 🗺️ Public roadmap and feature status
- 🔒 Security disclosure coordination
- 📄 Public documentation and FAQ

## What this repo is NOT

- ❌ Source code — frward is closed-source and proprietary
- ❌ Self-hosted deployment instructions — no frward self-hosted variant exists
- ❌ API documentation for the frward API — see [frward.xyz](https://frward.xyz) for the app
